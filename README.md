# 粤路慧养 v2.0 — 公路养护决策系统

基于《公路养护决策技术规范》(JTG/T) 构建的国省道养护决策分析平台。

## 安装运行

```bash
pip install pandas openpyxl numpy matplotlib
python main.py
```

可选：`psycopg2-binary`(PostgreSQL) `folium`(GIS地图)

## 系统架构

```
main.py                  # 主GUI（侧边栏导航，7步决策流程）
src/
  data_loader.py         # Excel加载/智能识别/字段默认值
  decay_calculator.py    # 衰减率标定/PQI预测
  database.py            # PostgreSQL/PostGIS数据库
  gis_map.py             # Folium GIS地图
  decision/
    performance_models.py   # 预测模型（指数/S形/马尔科夫）
    maintenance_demand.py   # 养护需求分析+优先级排序
    cost_model.py           # 费用模型+经济指标(B/C/LCC)
    budget_allocation.py    # 资金优化分配
    project_pool.py         # 工程项目库
    benefit_analysis.py     # 效益评估
```

## 决策流程（7 Tab）

### 1.数据管理
- Excel导入：2021-2025年数据，智能识别年份和Sheet
- 批量识别：选文件夹自动匹配年份
- 筛选：县份/年份/PQI等级/路面类型，支持导出
- 数据库：PostgreSQL/PostGIS连接与同步

### 2.现状评定
4张评价表 + 实时柱状折线图（带数据标注，可导出PNG/PDF）：
- 等级评价：国道/省道 × PQI/PCI/RQI，含优良路率/次差路率/分级里程
- 路线评价：按路线编码汇总加权均值与优良路率
- 技术等级评价：按等级分组，柱状均值+折线优良路率
- 年度趋势：跨年PQI/PCI/RQI均值与优良路率趋势

**分级**：PQI/PCI/RQI 优≥90 / 良≥80 / 中≥70 / 次≥60 / 差<60
**均值**：里程加权平均

### 3.目标设定
短/中/长期 × 技术+经济双维度：
- 技术：加权PQI、优良路率
- 经济：B/C比、每公里成本

对比表实时显示当前值与目标差距。

### 4.对策模型
- 衰减率标定：PQI(t)=PQI₀×e^(-k×t)，基于历年数据回归
- 触发阈值：路面改造/预防性养护条件，按路面类型×技术等级可调
- 回调值：养护后PQI/PCI/RQI回升目标
- **详细单价表**（17行，双击编辑）：沥青/水泥 × 结构性修复/功能性修复/预防养护 × 一/二/三级

### 5.需求分析
- 5年PQI预测 → 触发判断 → 养护类型 → 优先级排序 → 养护计划

### 6.投资规划
- 经济指标：加权PQI/B/C/LCC/单位成本
- 多年动态优化：年预算滑块(500-20000万) → 逐路段BCR排序 → 贪婪分配 → 逐年输出
- 项目库：导入导出/年度计划

### 7.GIS地图
Folium交互式地图，按PQI着色展示（绿→红）

## 核心算法

### 衰减率
PQI(t) = PQI₀×e^(-k×t)，线性化 ln(PQI/PQI₀)=-k×t 回归拟合

### B/C比
效益 = Σ(交通量×365×里程×PQI提升×0.015×车道数/10000)/(1+折现率)^t
B/C = 效益现值 / 养护投入

### 动态规划
```
每年：
  1.所有路段自然衰减 PQI×e^(-k)
  2.筛选 PQI<80 需改造路段
  3.每段BCR=(交通量×365×里程×ΔPQI×0.015×车道数)/费用
  4.按BCR降序贪婪分配预算
  5.选中路段PQI→92，剩余预算做预防性养护
  6.输出年末PQI/优良路率/累计效益
```

## 数据字段

| 字段 | 说明 | 默认值 |
|------|------|--------|
| 路线编码 | G/S/X开头 | - |
| PQI/PCI/RQI | 路面技术状况指数 | - |
| 路段长度km | 公里数 | 1 |
| 路面宽度 | 米 | 7 |
| 路面类型 | 沥青/水泥 | - |
| 技术等级 | 一/二/三/四级 | - |
| 交通量 | AADT辆/天 | 5000 |
| 车道数 | 1-4 | 2 |
| 路龄 | 年 | 5 |

## Excel格式

每年一个xlsx文件，按县份分Sheet，列名支持中英文自动识别。缺失字段自动填充默认值。
